# Icons

Drop real 16x16, 32x32, and 80x80 PNG icons here as `icon-16.png`, `icon-32.png`,
and `icon-80.png` before publishing to AppSource. `manifest.xml` references
them at `https://localhost:3000/assets/icon-*.png` for local dev; Excel will
show a generic default icon in the ribbon until these exist, which doesn't
block sideloading or testing.
